# Fluxo do Agente de IA

## Entrada
- Categoria
- Localização
- Palavras-chave
- Fontes

## Processo
1. Buscar notícias
2. Coletar dados
3. Filtrar duplicatas
4. Gerar rascunho
5. Sugerir mídia
6. Salvar draft

## Saída
- Draft com status pending_review
- Fontes registradas
- Score de confiança

## Regras
- Não inventar conteúdo
- Não publicar
- Registrar fontes
- Sugerir, não decidir
