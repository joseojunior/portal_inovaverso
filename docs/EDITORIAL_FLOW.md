# Fluxo Editorial

## Manual
1. Admin cria notícia
2. Salva rascunho
3. Edita conteúdo
4. Publica

## IA
1. IA pesquisa fontes
2. IA gera rascunho
3. Salva como draft_ai
4. Admin revisa
5. Admin aprova ou rejeita
6. Publicação manual

## Regras
- IA nunca publica
- Toda notícia precisa de revisão
- Toda notícia tem histórico de edição
